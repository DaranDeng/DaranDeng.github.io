# 回溯

## 组合问题
+ 本质上是无序的，分为题目所给原数组是否有重复元素
  + 无，题目会要求答案无重复的组合，比如（1，2）或（2，1）。startIndex 就是防止出现重复的组合。但是还分为：是否允许重复读取，就是组合内部是否可以有重复的元素。
    + 允许，递归传入i作为startIndex。
    + 不允许，传入i+1。
  + 有，**只要是原数组有重复元素，为了确保答案无重复的组合，无论是子集、组合还是排列，必须用树层去重加上对原数组排序。** used[i - 1] == true，说明同一树枝candidates[i - 1]使用过。used[i - 1] == false，说明同一树层candidates[i - 1]使用过。要对同一树层使用过的元素进行跳过：
    if (i > 0 && candidates[i] == candidates[i - 1] && used[i - 1] == false) {
        continue;
    }
